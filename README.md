# 🚜 Farm Tycoon Web Game

Selamat datang di **Farm Tycoon**, sebuah web-game interaktif dan responsif di mana Anda dapat membangun dan mengelola perkebunan impian Anda langsung dari browser!

## 🌟 Fitur Utama
- **Sistem Pertanian Real-Time**: Tanam berbagai bibit (Wortel, Jagung, Tomat, dll) dan tunggu hingga siap dipanen.
- **Sistem Hewan Ternak (Barn Area)**: Beli Ayam dan Sapi yang akan dipelihara di Kandang khusus! Hewan ternak ditampilkan menggunakan gambar animasi pixel-art yang lucu (bukan sekadar emoji).
- **Trio Kurcaci Pekerja (Auto-Farm)**: Bosan memanen manual? Sewa Kurcaci Petani, Peternak, dan Kurir untuk memanen, menanam, memungut susu/telur, dan mengirim pesanan secara otomatis!
- **Siklus Cuaca Dinamis**: Sistem cuaca acak (Cerah, Hujan, Badai) dengan efek animasi hujan turun yang realistis.
- **Dekorasi Premium**: Hias kebun Anda dengan Pohon, Bunga, Batu, hingga Air Mancur dan Patung.
- **Sistem Level & XP**: Kumpulkan XP untuk naik level dan membuka bibit serta hewan ternak baru.
- **Papan Pesanan (Order Board)**: Penuhi pesanan harian untuk mendapatkan hadiah Koin dan XP ganda!
- **Achievement & Quests**: Selesaikan berbagai misi harian dan raih trofi prestasi.
- **Efek Partikel & Visual Memukau**: Dapatkan kepuasan memanen dengan efek animasi partikel koin melompat, produk bersinar, dan UI bergradasi yang modern.

## 🚀 Cara Menjalankan Game
Game ini murni dibangun menggunakan HTML, CSS, dan JavaScript murni (Vanilla). Anda tidak perlu menginstal framework apapun!

1. **Clone/Download** repositori ini ke komputer Anda.
2. Buka file `index.html` langsung menggunakan web browser apa saja (Chrome, Safari, Firefox).
3. Mulailah bertani! Data permainan Anda (Koin, XP, Level) akan otomatis disimpan di browser Anda (LocalStorage), sehingga Anda bisa melanjutkannya kapan saja.

## 🛠️ Teknologi yang Digunakan
- **HTML5**: Struktur dan antarmuka UI.
- **CSS3**: Styling, animasi efek cuaca, efek partikel, dan Flex/Grid layout untuk *mobile responsiveness*.
- **JavaScript (ES6)**: Logika game (game loop, state management, timer).
